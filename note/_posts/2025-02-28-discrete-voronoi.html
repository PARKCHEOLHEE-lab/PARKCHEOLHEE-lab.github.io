  • Discrete Voronoi
    1. Creates a uniform grid
    2. Computes pairwise distances between sites and grid points
    3. Flips the sign to negative number so that the minimum distance is the maximum probability
    4. Converts distances to probabilities using softmax
    5. Assigns each grid point to nearest site using argmax
