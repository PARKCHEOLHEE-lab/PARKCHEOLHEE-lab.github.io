--- title: "Metrics" layout: post ---
  • Metrics for binary classification
    • Confusion Matrix

      • Confusion Matrix Predicted Positive Predicted Negative
      Actual Positive TP FN
      Actual Negative FP TN


    • Accuracy (정확도): 전체 예측한 것 중 올바르게 예측한 것의 비율. \[ \,\\ \text{Accuracy} = \frac{\text{TP} + \text{TN}}{\text{TP} + \text{TN} + \text{FP} + \text{FN}} \,\\ \]

    • Precision (정밀도): 모델이 참으로 예측한 것 중 실제로 참인 것의 비율. \[ \,\\ \text{Precision} = \frac{\text{TP}}{\text{TP} + \text{FP}} \,\\ \]
    • Recall (재현율): 실제 참인 것 중 모델이 참으로 예측한 비율. \[ \,\\ \text{Recall} = \frac{\text{TP}}{\text{TP} + \text{FN}} \,\\ \]
    • F1 Score: 정밀도와 재현율의 조화평균, 두 지표 간의 균형을 제공. \[ \,\\ \text{F1} = 2 \times \frac{\text{Precision} \times \text{Recall}}{\text{Precision} + \text{Recall}} \,\\ \]
