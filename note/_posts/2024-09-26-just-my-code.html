--- title: "JustMyCode configuration" layout: post ---
  • JustMyCode configuration of launch.json
    • It makes debugging the external module possible with "JustMyCode": false.
    • .vscode/launch.json is as: 
      • 
    {
        "version": "0.2.0",
        "configurations": [
            {
                "name": "Python Debugger: Current File",
                "type": "debugpy",
                "request": "launch",
                "program": "${file}",
                "justMyCode": false,
                "console": "integratedTerminal"
            }
        ]
    }

    • Before setting JustMyCode to false, the Step Into of Debugger is skipped.

    • After setting JustMyCode to false, the Step Into works well.
