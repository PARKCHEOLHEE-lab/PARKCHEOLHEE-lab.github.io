  • Claude Code Cheat Sheet

    • Slash Command Description
    /clear clear conversation history and start fresh
    /cost show token usage and cost for the current session
    /status show current session status info
    /config view or modify settings
    /usage view token usage
    /permissions view permission settings (Allow, Ask, Deny, Workspace)
    /plugin discover and install plugins
    /plugin install ralph-loop@claude-plugins-official install ralph-loop plugin
    /plugin install skill-creator@claude-plugins-official install skill-creator plugin
    /continue continue the conversation
    /skills list available skills (custom slash commands defined in .claude/skills/)

  • Bash Mode

    • Input Description
    !<command> run a single bash command directly (e.g. ! git status)
    ! (alone) enter bash mode — all subsequent inputs are executed as bash commands
    type any text in bash mode executed as a bash command without AI interpretation
    exit in bash mode return to normal Claude Code mode
