Objective

The purpose of this project is to develop a PoC by integrating the Landbook architectural engine with Hugging Face's Diffusers for generating architectural renderings. This PoC aims to: 1) Evaluate the feasibility of using Diffusers for rendering architectural outputs generated by the Landbook engine. 2) Experiment with prompt engineering to create workflows for producing consistent visualizations. 3) Explore the variability in results based on seed adjustments, negative prompts, and prompt refinements. This PoC explores the foundation for advancing AI-powered rendering solutions in architectural design.
